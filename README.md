# dollar-game

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build:prod
```

### Step one
```
Draw 3 circles in random positions
```

### Step two
```
connect them randomly
```

### Step three
```
Assign the three circles with a value between -1 & 9
```
